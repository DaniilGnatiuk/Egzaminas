# Egzaminas – Renginių sistema

## Projekto paleidimas

### 1. Backend (Node.js + Express)

**Paleidimas:**

```bash
cd BackEnd/express1
npm install
npm start
```

Serveris veiks ant `http://localhost:3000`

---

### 2. Frontend (React)

**Paleidimas:**

```bash
cd FrontEnd/frontend
npm install
npm run dev
```

Frontend veiks ant `http://localhost:5173` (arba kitame Vite porte)

---
